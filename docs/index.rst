Welcome to Charlesbot!
======================

Charlesbot is a modular Python bot written to take advantage of the Slack Real
Time Messaging API.

This website covers the project information for Charlesbot such as
installation, core development, plugin development, and so forth.

Please use `GitHub Issues`__ for filing any bug or feature requests, and `Pull
Requests`__ for submitting code patches.

__ https://github.com/marvinpinto/charlesbot/issues
__ https://github.com/marvinpinto/charlesbot/pulls

----

out!
